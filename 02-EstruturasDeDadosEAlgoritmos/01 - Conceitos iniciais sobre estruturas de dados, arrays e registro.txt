==========
O que é estrutura de dados
==========
> Estrutura de Dados: é uma estrutura organizada de dados na memória de um computador ou em qualquer dispositivo de armazenamento, de forma que os dados possam ser utilizados de forma correta,
Possuem muitas aplicações no desenvolvimento de sistemas, sendo que algumas são altamente especializadas e utilizadas em tarefas  específicas. Ao utilizar as estruturas adequadas através de algoritmos, podemos trabalhar com uma grande (ou pequena) quantidade de dados, como aplicações em bancos de dados ou serviços de busca.

Ao utilizarmos os algoritmos podemos manipular dados nas estruturas de várias formas, como por exemplo: inserir, excluir, procurar e ordenar dados.
Devemos então conhecer algumas estruturas de dados básicas, como por exemplo:
- Inserir Dados
- Excluir Dados
- Localizar um elemento
- Percorrer todos os itens constituintes da estrutura para visualização
- Classificar, que se resume em colocar os itens de dados em uma determinada ordem (numérica, alfabética, etc.)

Existem porem diversas estruturas de dados, as pricipais são:
- Vetores e Matrizes
- Registros
- Lista
- Pilha
- Fila
- Árvore
- Tabela Hash
- Grafos

==========
Vetores e matrizes
==========
Vetores e Matrizes, ou Arrays, são estruturas de dados simples que podem auxiliar quando há a necessidade de manipular muitas variáveis do mesmo tipo em um algoritmo.
https://portugol-webstudio.cubos.io/ide

Vetor ou Array unidimensional é uma variável que armazena várias variáveis do mesmo tipo. Tem por característica ser indexado, ou seja valores são enumerados/sinalizados, e que pode armazenar uma determinada quantidade de valores do mesmo tipo.

Matriz ou Array multidimensional é um vetor de vetores. É um vetor que possui duas ou mais dimensões.

==========
O que são registros
==========
> Registro: é uma estrutura que fornece um formato especificado para armazenar informações em memória. Enquanto Arrays nos permitem armazenar vários dados de um único tipo de dado, o recurso de Registro nos permite armazenar mais de um tipo de dado.
Um registro é composto por campos que especificam cada uma das informações. Ex.: Registro de um cliente possui campos como CPF, Nome, Valor de comprar no mês, Endereço e Contato.

Toda estrutura de registro tem um nome (Ex.: Nome), e seus campos podem ser acessados por meio do uso do operador ponto '.' .
Ex.: Para acessar o Endereço de um cliente:
    cliente.Endereço
