> Primeiros comandos com Git
===========
Iniciando o Git e criando um commit
===========
Objetivos:
- iniciar o Git
- Iniciar o versionamento
- Criar um commit

Comandos:
- git init : inicia o repositório no Git
- git add : mover/selecionar arquivos e dar início ao versionamento
- git commit : cria um commit
